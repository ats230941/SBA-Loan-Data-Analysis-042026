SBA Loan Performance Analysis (National)
🎯 Project Goal
To identify economic trends and default risks across 900,000+ small business loans. This project explores the relationship between loan characteristics (size, industry, location) and their eventual success or default status.

🛠 The Stack
Data Processing: Microsoft Excel & Power Query

Storage & Version Control: Local USB (Full Dataset) / GitHub (Documentation & Samples)

Project Management: Manual Time-Tracking & Phase Logging

📂 Data Attribution
Source: [SBA Loan Approval Dataset via Kaggle](https://www.kaggle.com/datasets/mirbektoktogaraev/should-this-loan-be-approved-or-denied)
License: CC0-1.0 (Public Domain)

Original Author: SBA / Kaggle Community

[!IMPORTANT]
Storage Note: Due to GitHub's file size limits for free accounts, the files hosted here are samples (first 100 rows). The full analysis is performed locally on the 900k-row master set.
